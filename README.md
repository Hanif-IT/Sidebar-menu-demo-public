# Futuristic Cars Sidebar Menu

A modern, responsive sidebar navigation component built with **HTML** and **CSS**. Features a dark theme with neon accent effects.

## 🎨 Features
- **Responsive Design:** Adapts to mobile and desktop screens.
- **Neon Aesthetics:** Custom CSS glow effects on the car and menu.
- **Smooth Transitions:** Hover effects on menu items (Gallery, Shortcuts, etc.).
